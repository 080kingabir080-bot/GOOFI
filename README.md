# SUS Games Hub - Full Platform

מערכת ניהול ומשחק בעיצוב עתידני.

## איך להשתמש:
1. העלה את שלושת הקבצים לתיקייה אחת ב-GitHub.
2. הפעל את ה-**GitHub Pages** דרך ההגדרות.
3. כנס לאתר:
   - כדי להיכנס לפאנל ניהול, התחבר עם שם המשתמש: **admin**.
   - כדי להוסיף משחק, פשוט הדבק קוד HTML/JS מלא לשדה הקוד.

## חוקי ברזל במערכת:
- **אין פסי גלילה:** האתר מותאם למסך מלא (Full Screen Viewport).
- **צד מנהל מובנה:** הוספת משחקים בלחיצת כפתור ללא עריכת קוד האתר.
- **עיצוב SUS:** שילוב של Glassmorphism, אנימציות כניסה ואפקטים של תאורה.
- **דירוג וביקורות:** מנוע פנימי מבוסס LocalStorage.
