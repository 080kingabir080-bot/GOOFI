# SUS Games Portal V2

## מערכת ניהול (Admin):
1. כדי להיכנס לניהול, הירשם עם שם המשתמש: **admin**.
2. המערכת תזהה אותך אוטומטית כמנהל ותוסיף כפתור "ניהול" בראש המסך.
3. בפאנל הניהול תוכל להוסיף משחקים (HTML/JS) ולמחוק משחקים קיימים.

## חווית משתמש:
- **דף הרשמה:** כולל מפלצת אדומה עם עיניים שעוקבות אחרי נקודות שונות במסך.
- **דף התחברות:** כולל מפלצת כחולה.
- **ללא גלילה:** האתר בנוי בטכנולוגיית `vh/vw` למניעת פסי גלילה מעצבנים.
- **אנימציות:** מעברים חלקים בין דפים (SPA).

## העלאה ל-GitHub:
פשוט צור Repository חדש, העלה את הקבצים, והפעל GitHub Pages תחת הגדרות ה-Settings.
